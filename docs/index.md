# EbonBuilds

<img src="https://raw.githubusercontent.com/Lzra2000/ProjectEbonHoldBuildAutomation/main/assets/banner.svg" alt="EbonBuilds — Echo automation for ProjectEbonhold" width="100%">

A World of Warcraft 3.3.5a addon for **ProjectEbonhold** that automates echo picks (Banish / Reroll / Freeze / Select) based on a build you define, and tunes itself over time from real gameplay data.

[Download the latest release](https://github.com/Lzra2000/ProjectEbonHoldBuildAutomation/releases/latest){ .md-button .md-button--primary }
[View on GitHub](https://github.com/Lzra2000/ProjectEbonHoldBuildAutomation){ .md-button }

Requires **ProjectEbonhold** or **ProjectEbonhold Enhanced**. Some features additionally use the **Details!** damage meter if installed.

## How it works

<img src="https://raw.githubusercontent.com/Lzra2000/ProjectEbonHoldBuildAutomation/main/assets/how-it-works.svg" alt="How it works: define a build, automation acts on every choice screen, data is tracked, the Tuning Advisor suggests adjustments, and the loop repeats" width="100%">

Define a build with per-echo weights and policies. Autopilot evaluates every echo choice screen against it and acts for you. Every decision lands in the Logbook with its reasoning; the Stats views aggregate the recorded runs; and the Tuning Advisor turns that evidence into concrete, reviewable suggestions.
