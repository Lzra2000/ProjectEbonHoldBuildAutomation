Welcome to the EbonBuilds wiki -- the documentation companion to the [repository](https://github.com/Lzra2000/-ProjectEbonHoldBuildAutomation).

EbonBuilds is a World of Warcraft 3.3.5a addon for **ProjectEbonhold** that automates echo picks (Banish / Reroll / Freeze / Select) based on a build you define, and tunes itself over time from real gameplay data.

**Pages**

- [[Getting Started]] -- install it, make your first build, turn on Autopilot
- [[Settings]] -- every option in the Settings window, category by category
- [[Localization]] -- how translations work and how to add a language
- [[Development]] -- contributor setup, the script toolbox, conventions
- [[Troubleshooting]] -- what to do when something doesn't work, and how to report it well
