## Bugs and feature requests

| Type | Link |
|------|------|
| Bug | [New bug report](https://github.com/Lzra2000/ProjectEbonHoldBuildAutomation/issues/new?template=bug_report.yml) |
| Feature | [New feature request](https://github.com/Lzra2000/ProjectEbonHoldBuildAutomation/issues/new?template=feature_request.yml) |

For bugs, attach **Error log** output (Settings → gear icon → Windows & tools). If a click did nothing, add the **Click Trace** log from the same panel. That is usually enough to fix something without back-and-forth.

## Security

Do not open a public issue for anything exploitable. See [`SECURITY.md`](SECURITY.md).
